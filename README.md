# Smart Digital Twin for Predictive Maintenance

This project demonstrates a Smart Digital Twin using real-time sensor simulation
and an Isolation Forest model to detect machine faults.

## Features
- Real-time sensor data simulation
- Anomaly detection using Isolation Forest
- Live dashboard using Streamlit

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run digital_twin.py
